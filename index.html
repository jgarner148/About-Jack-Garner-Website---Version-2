About Me

I'm a 19-year-old programmer who has just finished his first year studying Software Engineering at Edge Hill University. I have been programming since secondary school where I started learning Python3. Since the beginning of university, I have started to learn HTML and CSS as well as a bit of jQuery. These newly learned web coding skills are what I have used to make this website. Furthermore, I also began learning Java at Edge Hill which I am currently using in some of my projects. I would describe myself as having a deep-rooted passion for programming, computers, and how they work.

Childrens Story Box

This is a project I have been working on since the beginning of 2020 as something to do during the first Coronavirus lockdown. Since then I have finished the first version of the project and have started on the second

First Version

This version was written in Python3 and is text-based. The original aim with this version was to have an automatic and a manual mode that incorporated different versions of a basic AI. However, after the manual version was built the automatic version was not added and will instead be added to version 2.

Second Version

This version is being built in Java and is currently only in the early stages of development. The aim is for this to be a full graphical user interface with plans to make static cartoon images that will be collated into a comic-style page.
